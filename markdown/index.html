# Fron Matter

      Markdown 文件的 Front Matter 是用来配置页面的元数据或自定义配置的地方。Front Matter 位于 Markdown 文件的开头，使用三条短划线 (---) 包裹。在 Front Matter 中，可以定义各种选项，如页面标题、描述、布局、导航顺序等。

      Front Matter 一般使用 YAML 格式，同时它还支持其他语言格式，如 JSON 等，详见官方文档 (opens new window)。
